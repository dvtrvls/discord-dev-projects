# Dev Lab Repository

This repository is used by members of our Discord server to share programming projects, experiments, and learning resources.

We are learning programming by building things together.

## What goes here
- personal projects
- coding experiments
- small tools
- learning exercises

## Project Structure
python/
c/
web/
experiments/

Each project should have its own folder.

## How to contribute
1. Fork or clone the repository
2. Create a new folder for your project
3. Add a README describing your project
4. Push your code

## Requirements
You should know basic Git:
- clone
- pull
- commit
- push

If you don't know Git yet, ask in the Discord server.
