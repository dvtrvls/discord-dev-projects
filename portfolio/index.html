I am a first‑year Applied Mathematics student with a strong desire to integrate mathematics and technology. I began learning to code just two months before turning 18, and like many beginners who self‑study, I spent a long period feeling directionless. For nearly ten months I scattered my efforts across multiple languages and fields — Python, C, C++, Java, JavaScript, HTML, CSS, networking, cybersecurity, arduino, and even assembly — without gaining more than surface‑level knowledge in any of them. I carried the misconception that to be a “tech person” I had to know everything at once, which left me unfocused and overwhelmed.

Since February, however, I’ve shifted my approach. I chose to concentrate on one path, and in just the past two weeks I’ve already built small projects in Python. Now I dedicate time each day to strengthening my Python skills, exploring web development, and gradually diving into low‑level programming. With this renewed focus, I feel more grounded, motivated, and confident that I’m building a solid foundation for my future in both mathematics and technology.

MY PROJECTS

  • Data Science

    I did several data science and machine learning projects during January and February. While most of them were done by following step-by-step tutorials, I was still exposed to many important concepts and tools that helped me build a foundational understanding of the field.

    During this time, I worked on projects such as linear regression model and probability-function based model ,including simple prediction tasks, and the Titanic survival prediction project, where I learned how to handle real-world datasets. I used libraries like NumPy for numerical operations, pandas for data manipulation and cleaning, and matplotlib for basic data visualization. I also explored different machine learning models using scikit-learn, where I learned how to train models and improve their accuracy.

  • Bulacan Municipality Completing Quiz

    This game is inspired by the popular U.S. quiz where players name the states, but instead of focusing on the United States, I built my version around Bulacan, where I’m from. This gave the project a personal twist while still following the same engaging format. Working on it taught me valuable skills: I learned file manipulation by storing and reading data about places, practiced using the Turtle module to create interactive visuals, and strengthened my logical thinking by designing the rules, checking answers, and handling user input.

  • Pomodo GUI

    I created a Pomodoro‑style timer that helps me manage work and break sessions. Building it taught me several important skills: I used pymixer in pygame to produce sound notifications, practiced the logic needed to make the clock count down and automatically switch between work and break modes, and explored Tkinter to add a background picture for a more polished interface. hrough this project, I strengthened my understanding of how different Python modules can work together, while also improving my a bility to design functional and visually engaging applications.

  • Turtle module based games

    I created several different games, including a Snake game, a Ping Pong game, a Turtle Race game, and a Turtle Crossing game. Most of these projects relied heavily on the **Turtle module**, which allowed me to build interactive graphics and animations. Working on them enhanced my **logical thinking**, since I had to design rules, handle collisions, manage scoring, and control movement patterns. Each game challenged me to break problems into smaller steps and apply programming concepts creatively, helping me grow both as a coder and as a problem solver.
